# GU_Bioinformatics_26
##This is the **readme** file! 
This repository contains the materials for our spring 2025 Biol 3655 class. In this class we are working through a metagenomics project, identifying the DNA viruses within different permafrost samples from the peatland in Northern Sweden. The publication we are referencing is: 
Trubl G, Jang HB, Roux SEmerson JBSolonenko N, Vik DR, Solden L, Ellenbogen J, Runyon AT, Bolduc B, Woodcroft BJ, Saleska SR, Tyson GW, Wrighton KCSullivan MB, Rich VI. 2018. Soil Viruses Are Underexplored Players in Ecosystem Carbon Processing. mSystems 3:10.1128/msystems.00076-18.
https://doi.org/10.1128/msystems.00076-18
